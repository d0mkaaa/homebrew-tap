# homebrew-tap

Homebrew tap for [gopull](https://github.com/d0mkaaa/gopull).

## Install

```sh
brew install d0mkaaa/tap/gopull
```

## Upgrade

```sh
brew upgrade d0mkaaa/tap/gopull
```

## Note on macOS security

gopull is not signed with an Apple Developer ID. If macOS blocks the binary
on first run, remove the quarantine flag:

```sh
xattr -d com.apple.quarantine "$(which gopull)"
```
